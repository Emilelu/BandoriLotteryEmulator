# 注意事项
编译时，确保 Classpath 内包含 jlx.x.x.jar, mp3spix.x.x.jar, tritonus_share.jar，否则音乐播放类会报错，但不影响主程序的运行。  
建议编译为可执行的 JAR 文件，并将 bgm.mp3 与 JAR 文件放在同一目录中。  
不建议在 IDE 中直接 Run，否则你需要修改 Music.java 中 bgm.mp3 的路径。
